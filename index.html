Surgimento do HTML

(voltar)

O HTML (HyperText Markup Language) surgiu no início da década de 1990, criado por Tim Berners-Lee, um cientista britânico que trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear), na Suíça.

Na época, Tim queria facilitar o compartilhamento de documentos científicos entre pesquisadores de diferentes lugares do mundo. Para isso, ele criou um sistema que permitia interligar documentos através de links, ou seja, o hipertexto.

Em 1991, ele publicou a primeira versão do HTML, que continha cerca de 20 tags básicas. Esse sistema se tornou a base da World Wide Web (WWW), ou simplesmente, a internet como a gente conhece hoje.

O HTML permitia que as páginas fossem escritas e lidas de forma padronizada, possibilitando que qualquer pessoa, em qualquer computador, conseguisse acessar e visualizar os mesmos conteúdos.

Desde então, o HTML passou por várias evoluções. Hoje estamos na versão mais recente, o HTML5, que trouxe várias melhorias como suporte a vídeos, áudios e novos elementos para tornar os sites mais modernos e interativos.

Introdução

(voltar)

O HTML (HyperText Markup Language) é a linguagem usada para estruturar páginas na internet. Ele funciona através de tags, que são comandos envoltos por sinais de menor e maior que (< >). Essas tags indicam como o conteúdo deve ser exibido no navegador. Abaixo, explico as principais:

  • HTML

    É a tag raiz de todo documento HTML. Tudo o que for exibido na página deve estar dentro dela.

  • head

    Dentro dessa tag ficam informações não visíveis diretamente na página, como título, links de estilos (CSS), scripts (JavaScript) e metadados.

  • title

    Define o título da página, aquele que aparece na aba do navegador.

  • Body

    Tudo que o usuário vê na página (textos, imagens, botões).

  • h1 até h6

    São as tags de cabeçalho ou títulos, variando da maior importância (h1) até a menor (h6).

  • p

    Usada para criar parágrafos de texto.

  • a

    Cria um link (âncora) para outra página ou site.

  • img

    Exibe uma imagem na página.

  • ul, ol, il

    Criam Listas:

    • ul = listas não ordenadas (com bolinhas).

    • ol = lista ordenada (com números).

    • li = item da lista.
